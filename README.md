## ‚ú® What is Project 365?

Project 365 is a Notion-like planning web app that helps you:

- Define a goal (fitness, business, learning, creative, etc.)
- Timebox it to ‚â§ 365 days
- Let AI design a detailed plan with milestones and daily actions
- Track your progress, adapt the plan, and ‚Äúfeel the clock‚Äù as the deadline approaches

The core idea: if you follow the plan exactly, it should be **unreasonable** for you to fail.

You bring the goals and discipline.  
The app brings structure, visibility, and relentless reminders.

## üåê Web-Only Architecture (for now)

Right now, Project 365 is being built as a **web-only application**:

- Built with **React + TypeScript** and **Vite** as the bundler.
- Designed primarily for a **desktop browser** experience (laptop/monitor viewport).
- Uses **Tailwind CSS** for styling and a clean design system.
- Future directions (native desktop wrappers, mobile) can reuse the same core logic and components, but all initial effort is focused on the browser.

## üß© Key Concepts

- **Projects** ‚Äì Each project is a goal with a start date and a target date (up to 365 days).
- **Milestones** ‚Äì Monthly/quarterly checkpoints that break the journey into chunks.
- **Daily Tasks** ‚Äì Simple, check-off-able actions generated by AI.
- **Missed Tasks** ‚Äì Tasks you didn‚Äôt complete on time move into a weekly ‚Äúmissed‚Äù bucket.
- **Days Remaining / Days Behind** ‚Äì Constant feedback on how close you are to the deadline and how far behind (or ahead) you are.

Think of it as a focused, goal-specific workspace that behaves like a living document.

## üß† AI-Assisted Planning

Project 365 uses the OpenAI API (with **your** API key) to:

- Generate an initial, fully structured plan for each project:
  - Milestones and checkpoints
  - Daily/weekly routines
  - Realistic intensity based on your constraints
- Continuously adjust the plan when you:
  - Change the deadline
  - Fall behind schedule
  - Decide to make the goal more ambitious
- Rewrite, re-scope, or completely replace the plan on command

You‚Äôll be able to:

- Ask the AI: _‚ÄúShorten this goal to 90 days.‚Äù_
- Ask: _‚ÄúI missed most of last week. Replan the next two weeks.‚Äù_
- Ask: _‚ÄúI‚Äôm ahead of schedule. Make this goal harder.‚Äù_

All project state is visible and editable ‚Äì the AI just operates on the same data model you see.

## üëê Open Source Philosophy

Project 365 is designed to be:

- **Fully open source** ‚Äì you can audit, fork, extend, and self-host.
- **Locally respectful** ‚Äì the OpenAI API key is provided by you and used directly from your app.
- **Extensible** ‚Äì the domain model, AI prompts, and UI are intentionally modular:
  - Want a different AI provider? Swap the client.
  - Want new goal types or metrics? Extend the domain layer.
  - Want custom visualizations? Build on top of the existing components.

Community contributions are welcome once the core architecture stabilizes:
- New views/components
- Better prompts and schemas
- Integrations (calendar, wearable data, etc.)
- Documentation and examples
